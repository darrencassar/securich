Baby proof install script readme:

tar -xf securich_install.tar
chmod 700 securich_install.sh

Steps involved
1. enter version number when enquired "obtainable from http://www.securich.com/downloads.html"
2. choose from install from file or download and install (recommended)
2. choose fresh install or upgrade
3. enter mysql root password when prompted
4. choose weather to connect using the socket or over tcp
5. enter mysql hostname and port OR socket when prompted



*********************************************************************
**** Make sure you have mysql, wget, gunzip and tar in your PATH ****
*********************************************************************

enjoy!
